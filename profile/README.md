## About

Sparkware is an independent software company focused on building lightweight, well-crafted apps for Android, Windows, and the web. We care about user experience, performance, and keeping things simple.

Everything we ship is made with attention to detail — from first launch to long-term use.

## What We Build

| Platform | Focus |
|----------|-------|
| 🤖 Android | Native apps with clean UI and good performance |
| 🪟 Windows | Desktop software that doesn't get in your way |
| 🌐 Web | Fast, accessible websites and web apps |

---

## Philosophy

- **Simple over clever** — if it needs a manual, it needs a redesign
- **Ship what works** — working software beats perfect software
- **Respect the user** — no dark patterns, no unnecessary permissions, no noise

## Getting Started

Each project has its own setup instructions in its repo. Head to the specific project README for installation steps, requirements, and usage.

Most of our tools follow a standard pattern:

```bash
# Clone the repo
git clone https://github.com/sparkware/<project-name>

# Follow the project-specific setup
cd <project-name>
```

## Contributing

We welcome contributions — bug reports, feature suggestions, or pull requests.

1. Fork the repo you want to contribute to
2. Create a branch (`git checkout -b fix/your-fix`)
3. Commit your changes (`git commit -m 'fix: short description'`)
4. Open a pull request

Please keep PRs focused and scoped. One thing at a time.
